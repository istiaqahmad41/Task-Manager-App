# Simple Task Manager App (Flutter + Hive)

A basic and lightweight To-Do app built with **Flutter** and **Hive** for local data storage. This app allows users to **add** and **delete** tasks with a minimal and clean interface ‚Äî perfect for learning local persistence in Flutter.

---

## Features

-  Add new tasks
-  Delete existing tasks
-  Local storage using [Hive](https://pub.dev/packages/hive)
-  Clean and responsive UI
-  Fast and offline-ready
